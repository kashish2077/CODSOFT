# CODSOFT Machine Learning Internship

Welcome to my CodSoft Machine Learning Internship repository. This repository contains the projects completed during my internship, demonstrating various Machine Learning concepts such as classification, prediction, and fraud detection.

## Projects

### 1. Titanic Survival Prediction

* Built a classification model to predict passenger survival on the Titanic.
* Algorithm Used: Random Forest Classifier
* Accuracy Achieved: 82.12%

### 2. Iris Flower Classification

* Developed a model to classify Iris flowers into different species.
* Applied supervised machine learning techniques.

### 3. Credit Card Fraud Detection

* Built a fraud detection system to identify fraudulent transactions.
* Focused on handling imbalanced datasets and classification.
