## 📂 Projects Completed

### 🚢 Titanic Survival Prediction

Predict whether a passenger survived the Titanic disaster based on demographic and travel information.

**Skills Used:**

* Data Cleaning
* Feature Engineering
* Label Encoding
* Random Forest Classification

**Result:** Achieved **82.12% Accuracy**

---

### 🌸 Iris Flower Classification

Developed a machine learning model to classify Iris flowers into different species based on flower measurements.

**Skills Used:**

* Data Visualization
* Classification Algorithms
* Feature Analysis

### 💳 Credit Card Fraud Detection

**Skills Used:**

* Classification Models
* Performance Evaluation
